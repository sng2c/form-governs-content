> 이 문서가 현재 산출물의 **정합(canonical) 뷰**. 실험 *데이터/측정값은 변하지 않으며*, 해석의 틀이 v0.3.0(정합적 이상 · MVP=제어판 · 환상 컨트롤 삭제)로 정렬됨. 과거 세션 기록은 [`PRUNER_LOG.md`](PRUNER_LOG.md) 참고.

## 1. 정합적 이상 $I_0$ (존재 여부 미예단)
**"완전 자동 · 비용 0 · 모델 무관한 창의적 종합 파이프라인"** — 목적+내용을 넣으면 *진짜* 창의적 종합이 나오고, 인간·대형 모델·클라우드 전부 불필요.

$I_0$는 **정합적 이상**이다(실재 여부 예단 금지). 소멸 테스트가 $I_0$를 **하향 정정** 중:
- "창의적 종합" 부분은 *소멸*(형식 주도·자동화 가능).
- "완전 자동"은 $\alpha$(진정성)가 소멸할 때만 성립. $\alpha$가 저항하면 $I_0$(완전 자동)는 **환상**이고, 진짜 산출물은 "자동 코어 + 최소 제어판".

## 2. 제어판(후보 컨트롤) 소멸 지도
세션에서 연기된 블랙박스 전부를 후보 컨트롤로 재해석. 각 컨트롤을 하강에서 테스트해 **소멸(환상→삭제) / 저항(실재→유지) / 미테스트(프론티어)** 로 분류:

| 후보 컨트롤 (블랙박스) | 소멸 상태 | 근거(실험) | 제어판 처분 |
|---|---|---|---|
| $\Phi_0$: 이상 템플릿 *생성* $T^*(P,M)$ | **소멸** | 형식이 구조·종합을 모델 무관 보장(Exp1/2/3 마커 1.0, 1B Aufhebung) → 템플릿은 *불변* | **삭제(환상)** |
| $\Phi_{\text{route}}$: 모델 선택 $M^*(P)$ | **대부분 소멸** | synthesize/surface는 모델 무관(Exp3 1B OK) → 라우팅 매트릭스(선형)로 흡수; 잔여 = decide 쟁점 판정만 모델 의존(Exp2) | 매트릭스화(자동); 잔여 → `intent` |
| $\Psi$: 최적 특징 벡터 $F^*$ | **소멸** | 템플릿 불변이면 특징 선택 무의미 → $\Phi_0$ 소멸에 흡수 | **삭제(환상)** |
| `intent`: decide 쟁점의 편향 방향 | **소멸 저항(실재)** | "어느 bias를 원하는가"는 인간 가치 판단, 자동화의 정답 없음(Exp2 모델별 support/oppose 관측) | **유지(실재, 스위치)** |
| $\alpha$($J_{\text{auto}}$): 종합 진정성(지양 vs novelty-cheat) | **미테스트(프론티어)** | 1B가 높은 novelty 내지만 치트일 수 있음; $\alpha\approx f(m,\rho,\nu,\kappa_{\text{op}})$ 잔여 미측정 | **후보, 소멸 테스트 대기** |
| $\Sigma$: 지양 판별 | $\alpha$의 한 측면으로 흡수 | 진정성 = 지양 $\wedge$ concreteness | (별도 컨트롤 아님) |

## 3. 현재 MVP = 현재 제어판 (동결)
환상 컨트롤을 삭제한 뒤 살아남은 제어판:

**자동 코어(소멸/선형, 인간 컨트롤 없음):**
$$P \xrightarrow{\text{고정 형식}} F \xrightarrow{\text{라우팅 매트릭스}} M \xrightarrow{\text{종합 생성}} S_3$$
1B 비-reasoning까지 robust → 비용 0·모델 무관, 이미 자동화됨. (구현: [`ROUTING.ko.md`](ROUTING.ko.md))

**컨트롤 #1 — `intent` (실재, 유지):** decide 쟁점 항목에서 편향 방향(보수 검증 vs 점진 포용)을 기획자가 고르는 스위치. 인간 책임. 자동화 불가(정답 없음).

**컨트롤 #2 — $\alpha$ 진정성 게이트 (후보, 미테스트):** $S_3$가 진짜 지양인지 novelty-cheat인지. 프록시(novelty $\nu$, operational concreteness $\kappa_{\text{op}}$)가 후보를 좁히고 인간은 경계선만 판정.
- **I/O 규격:** 입력 $(S_3, P, M, \text{관측량 } m,\rho,\nu,\kappa_{\text{op}})$ → 출력 {authentic / borderline→인간 / not}.

> v0.3.0의 핵심: 이 MVP가 곧 *현재 제어판*. 환상 컨트롤($\Phi_0$, $\Phi_{\text{route}}$ 대부분, $\Psi$)은 삭제됐고, 실재(`intent`)는 유지, 후보($\alpha$)는 소멸 테스트 대기. 제어판은 하강에서 환상 컨트롤을 삭제하며 줄어든다.

## 4. 프론티어 — $\alpha$ 소멸 테스트 (다음 하강)
$\alpha \approx f(m,\rho,\nu,\kappa_{\text{op}})$의 잔여를 측정(raw 3종 보존 완료 → 가능).
- **잔여 → 0**: $\alpha$는 *소멸*(환상) → 제어판에서 삭제 → $I_0$(완전 자동) 회복.
- **안정 잔여 + 인간 합의 일치**: $\alpha$는 *실재* → 제어판에 유지 → $I_0$(완전 자동)는 환상 확정, 진짜 산출물 = "자동 코어 + 2-컨트롤 제어판".

## 5. 각 실험의 v0.3.0 독해 (어떤 컨트롤을 소멸/확정했나)
- **Exp1**(120, 단일모델·8형식): 형식이 내용을 지배(verdict 분산 0.283) → "$\Phi_0$: 템플릿이 내용과 무관"이라는 숨은 가정을 *정정*(형식이 판정을 결정). triad가 이항 거부+제3 종합 → 창의성=형식 주도의 첫 증거.
- **Exp2**(96, 8모델·목적별): synthesize/surface의 구조·종합이 모델 무관 → $\Phi_{\text{route}}$(모델 선택) *대부분 소멸*. 단 decide 쟁점 판정은 모델 의존(support/oppose 분기) → `intent` 컨트롤 *실재 확정*.
- **Exp3**(75, 1–12B 로컬·triad): 1B 비-reasoning까지 Aufhebung(이항거부 1.0, novelty 0.96–0.995) → "대형/reasoning 필요" 컨트롤 *소멸*(환상); 창의성=형식 주도 *확정*. 단 1B novelty-cheat 위험이 $\alpha$ *후보 컨트롤*을 부상시킴.
- **ROUTING**(매트릭스): 자동 코어의 구현 + `intent` 컨트롤의 구체화. $\Phi_{\text{route}}$의 소멸된 부분 + 저항한 잔여(intent)가 함께 살아 있음.

## 6. 산출물 상태 (v0.3.0 기준)
- **자동 코어**: 구현됨(ROUTING 매트릭스 + 1B 로컬 검증).
- **컨트롤 #1 intent**: 실재 확정, 스위치로 명문화 가능.
- **컨트롤 #2 $\alpha$**: 후보, 소멸 테스트가 다음 단계(raw 보존 완료로 unblocked).
- **$I_0$(완전 자동)**: 정합적 이상으로 보류, $\alpha$ 테스트가 실재/환상 결정.
